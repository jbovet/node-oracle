
# node-oracle

Docker image for using instant client v11.2.0.4.0 with Node.js 9

## Deps

- Instant Client Basic Linux
- Instant Client SDK Linux
- Node.js 9

## How to Use

```
  docker build . -t node-ora
```  

## Building

```
  docker build . -t node-ora
  docker run -it node-ora /bin/bash
```
